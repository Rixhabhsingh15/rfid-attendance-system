# IoT-Based RFID Attendance System

This project is an IoT-based attendance system using RFID technology and Arduino.
Attendance data is automatically recorded and updated in Google Sheets.

## Features
- RFID-based student identification
- Automatic attendance logging
- Real-time Google Sheets update

## Technologies Used
- Arduino
- RFID Module
- Google Sheets
